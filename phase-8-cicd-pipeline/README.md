# Phase 8 — CI/CD Pipeline

This phase implemented a continuous integration and deployment pipeline using AWS CodePipeline and CodeDeploy.

The pipeline automates the process of building, testing, and deploying updates to the microservices.

## Tasks Completed

- Created a CodeCommit repository
- Configured CodePipeline stages
- Integrated CodeDeploy for deployment automation

The pipeline enables automated deployments whenever changes are pushed to the repository.
