# Phase 6 — Application Load Balancer

This phase involved creating an Application Load Balancer (ALB) to route traffic to the correct microservice containers.

The ALB acts as the public entry point for the system and distributes requests to the correct ECS services using routing rules.

## Tasks Completed

- Created target groups for each microservice
- Configured listeners and routing rules
- Verified traffic routing

The load balancer was successfully configured and prepared to route requests between services.
