# Task 4.3 — Create Employee Microservice

## Objective

Create a separate microservice responsible for supplier management and administrative features.

## Actions Performed

- Created a new employee service directory
- Moved supplier management functionality into the employee service
- Included routes for adding, editing, and deleting suppliers
- Ensured the service could run independently from the customer service

## Result

The employee/admin functionality was separated from the monolithic application and placed into its own microservice.
