# Task 4.2 — Create Customer Microservice

## Objective

Create a dedicated microservice for the customer-facing supplier pages.

## Actions Performed

- Created a separate service structure for the customer application
- Moved supplier viewing functionality into the customer service
- Ensured only public supplier routes were included
- Prepared the service to run independently from the admin features

## Result

The customer-facing part of the application was successfully separated from the monolith into its own service.
